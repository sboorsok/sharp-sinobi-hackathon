1. ТАБЛИЦА (БД-ЗАДАЧИ)
* ID                                            BIGINT
* Наименование задачи          NVARCHAR(200)
* Описание задачи                NVARCHAR(3000)
* Дата выполнения                DATETIME
* Приоритет (важно, не важно)   BIT (bool)
* Выполнено \ в процессе \ просрочено \ отложено ENUM
2. МОДЕЛИ (Класс в отдельной папке в визуал студио)
* ID                                LONG
* Наименование задачи        STRING
* Описание задачи                STRING
* Дата выполнения                DATETIME
* Приоритет (важно, не важно)    BOOL
* Выполнено \ в процессе \ просрочено \ отложено ENUM

* ChangeStatus()
* GetSortedTasks() (по дате выполнения и приоритету)
